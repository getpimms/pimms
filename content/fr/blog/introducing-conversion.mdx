## 🎯 Fini les suppositions : passe à la clarté

Pose-toi (et réponds à) ces questions :

Avec PiMMs, ces réponses sont automatiques — sans tableur, sans dev complexe.

## ⚙️ Comment fonctionne le suivi PiMMs

Chaque lien court PiMMs contient un `pimms_id`. Ce code est transmis automatiquement du clic → au formulaire → à Zapier ou à ton serveur.

Tu n’as besoin que de deux événements :
- **Click** : Un visiteur clique sur un lien PiMMs.
- **Conversion** : Tu déclares une conversion (via Zapier ou ton backend).

## 🔌 Options d’intégration

### Intégration site web (sans code)

Ajoute ce code dans le `<head>` de ton site :

```html
<script defer src="https://cdn.pimms.io/analytics/script.detection.js"></script>
```

:::info
**Vérifie que tout fonctionne (Optionnel)**

Ouvre ton site et écris `window._pimmsAnalytics` dans la console du navigateur.  
Si tout est bien installé, des valeurs s’afficheront.
:::


Ajoute `?pimms_id=1` (ou `?pimms_client_reference_id=1` pour Stripe) à tes liens internes quand tu ne peux pas utiliser un lien court PiMMs.

**Exemples :**

**Cal.com — Liens de rendez-vous (lead tracking)**
```html
<a href="https://cal.com/ton-nom/meeting?pimms_id=1">Réserver</a>
```

**iClosed — Formulaire de deal**
```html
<a href="https://app.iclosed.io/e/xyz?pimms_id=1">Soumettre un deal</a>
```

**Stripe — Lien de paiement (sales tracking)**
```html
<a href="https://buy.stripe.com/xyz?pimms_client_reference_id=1">Acheter</a>
```

💡 PiMMs injectera dynamiquement le bon `pimms_id` ou `client_reference_id` selon l’origine du visiteur — attribution garantie.

### Intégration React (client-side)

```jsx
import { Analytics as PimmsAnalytics } from "@getpimms/analytics";

<PimmsAnalytics
  cookieOptions={{ domain: ".tondomaine.com" }}
/>
```

### Intégration backend (suivi serveur)

```ts
import { Pimms } from "pimms";

const pimms = new Pimms({ token: "YOUR_TOKEN" });

await pimms.track.lead({
  eventName: "Inscription",
  externalId: "user_123",
  customerEmail: "jane@example.com",
  customerName: "Jane Doe",
  metadata: {
    campaign: "Semaine de lancement",
    source: "Post LinkedIn",
  },
});

res.clearCookie("pimms_id");
```

### Backend (server-side) sales tracking

```ts
import { Pimms } from "pimms";

const pimms = new Pimms({ token: "YOUR_TOKEN" });

await pimms.track.sale({
  amount: 100,
  currency: "USD",
  eventName: "Purchase",
  externalId: user.id,
  paymentProcessor: "stripe",
});

res.clearCookie("pimms_id");
```

Ou utilise Zapier avec des outils comme **Cal.com**, **Tally.so**, ou **iClosed** pour envoyer les conversions automatiquement.
