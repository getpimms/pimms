Tu veux savoir précisément d’où viennent tes leads et tes ventes sur Systeme.io — même dans un funnel à plusieurs étapes ?

Avec **PiMMs**, ça ne prend que quelques minutes — sans code.

## 1. Installer le script de suivi

1. Va dans ton **tableau de bord Systeme.io**

![Paramètres Systeme.io — section tracking](https://assets.pimms.io/systemeio-footer-code-settings.webp?v=2)

2. Ouvre les **paramètres du profil** (Profile Settings)
3. Dans le menu de gauche, sélectionne **Sales funnels**
4. Colle ce script dans le champ **Tracking code** :

```html
<script defer src="https://cdn.pimms.io/analytics/script.detection.js" data-forward-all="true"></script>
```

5. Clique sur **Enregistrer**

## 2. Trouver ton Workspace ID

1. Connecte-toi à ton [dashboard PiMMs](https://app.pimms.io)  
2. Va dans **Settings → Workspace ID**  
3. Copie ton ID (exemple : `ws_abc123xyz`)

## 3. Ajouter le webhook

1. Dans Systeme.io, clique sur ta photo → **Paramètres**  
2. Descends jusqu’à la section **Webhooks**  
3. Clique sur **Créer un webhook**  
4. Configure-le ainsi :

- **Nom** : PiMMs Webhook  
- **URL du webhook** :  

```
https://app.pimms.io/api/systemeio/webhook?workspace_id=ton-id
```

*(remplace `ton-id` par ton Workspace ID)*  
- **Secret** : ton Workspace ID  
- **Événements à activer** :  
  ✅ `Opt-in`  
  ✅ `New sale`

![Paramètres du webhook](https://assets.pimms.io/systemeio-webhook-settings.webp)

⚠️ L’URL et le secret doivent **absolument contenir le même Workspace ID**.

## 4. Créer un lien de suivi dans PiMMs

1. Connecte-toi à [PiMMs](https://app.pimms.io)  
2. Crée un nouveau **lien court** vers la **première étape** de ton funnel Systeme.io  
3. Active l’option **Conversion tracking**

### Exemple de lien court

```
https://pim.ms/mon-funnel
```

👉 Utilise ce lien dans toutes tes sources de trafic : publicités, emails, réseaux sociaux…

![Lien court dans PiMMs](https://assets.pimms.io/systemeio-create-short-link.webp)

## 5. Tester ta configuration

1. Ouvre ton lien PiMMs en **navigation privée**  
2. Suis toutes les étapes de ton funnel (opt-in + vente)  
3. Utilise un email de test (ex : `test+pimms@tondomaine.com`)  
4. Vérifie ton [dashboard PiMMs](https://app.pimms.io)

Tu dois voir apparaître :

- ✅ Un **lead**
- ✅ Une **vente** (si applicable)

![Dashboard PiMMs avec lead et vente](https://assets.pimms.io/systemeio-dashboard-lead-sale.webp)

## 6. Conseils en cas de problème

- Assure‑toi que le script PiMMs est bien enregistré dans les paramètres du funnel
- Vérifie que le **Workspace ID** est identique dans l’URL et dans le secret du webhook
- Refais un test en **navigation privée** avec une nouvelle adresse email
- Consulte les logs des webhooks dans Systeme.io pour vérifier que les événements ont bien été déclenchés
- Assure‑toi que les visiteurs passent bien par ton **lien PiMMs** (et pas directement)
